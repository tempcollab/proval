## Full proof

Throughout, ℕ₀ = {0, 1, 2, …}. For a positive integer n we write
rad(n) for its **radical**, the product of the distinct primes dividing n (with
rad(1) = 1, the empty product). Equivalently, rad(n) encodes the *set* of prime
divisors of n. For a prime p and a nonzero integer x, v_p(x) denotes the exponent
of p in x.

### Preliminary: companions ⇔ equal radicals

**Claim.** For a, b ∈ ℕ₀:

- If a, b ≥ 1, then a and b are companions if and only if rad(a) = rad(b).
- 0 is a companion of a number c if and only if c = 0.

**Proof.** By definition a and b are companions iff every prime divides both or
neither, i.e. iff a and b have exactly the same set of prime divisors. For
positive integers, having the same set of prime divisors is precisely rad(a) =
rad(b). For the degenerate case: every prime divides 0, so 0 is a companion of c
iff every prime divides c, which forces c = 0 (a positive integer has only
finitely many prime divisors). ∎

We will apply the functional equation (FE) only at arguments where the two sides
are positive, except for the single value f(0) + 0 = 0, which is a companion of
0 + 0 = 0, consistent with the FE. Every rad(·) = rad(·) equality we invoke below
has both arguments ≥ 1, so it is honest.

### Step 1: rad(f(n)) = rad(n) for all n, and f(n) = 0 ⇔ n = 0

Put m = 0 in the FE. The two companions are f(0) + n = 0 + n = n and
f(n) + 0 = f(n). Hence n and f(n) are companions for every n ∈ ℕ₀.

For n = 0: f(0) = 0 = n, consistent (0 companion of 0).

For n ≥ 1: n is positive, so by the Preliminary its companion f(n) must also be
positive (0 is a companion only of 0), and rad(f(n)) = rad(n). In particular a
prime divides f(n) iff it divides n. Conversely if n = 0 then f(0) = 0.
Thus **f(n) = 0 ⇔ n = 0**, and **rad(f(n)) = rad(n)** for all n. ∎

### Step 2: f(1) = 1

By Step 1, rad(f(1)) = rad(1) = 1, so f(1) has no prime divisors. Since f(1) ≥ 1
(as 1 ≥ 1, by Step 1 f(1) ≠ 0), the only positive integer with empty set of prime
divisors is 1. Hence **f(1) = 1**. ∎

### Step 3: rad(f(m) + 1) = rad(m + 1) for all m

Put n = 1 in the FE. The companions are f(m) + 1 and f(1) + m = 1 + m (using
Step 2). Both arguments are ≥ 1 (indeed ≥ 1, and ≥ 2 for m ≥ 1). By the
Preliminary, **rad(f(m) + 1) = rad(m + 1)** for all m ∈ ℕ₀. ∎

### Step 4: f(p) = p for every ODD prime p

Fix an odd prime p. By Step 1, rad(f(p)) = rad(p) = p, so f(p) is a positive
integer whose only prime divisor is p; therefore

  f(p) = p^k  for some integer k ≥ 1.

By Step 3, rad(f(p) + 1) = rad(p + 1), i.e.

  rad(p^k + 1) = rad(p + 1).   (★)

We prove **Lemma A**: for an odd prime p and integer k ≥ 2 one has
rad(p^k + 1) ≠ rad(p + 1). Given the lemma, (★) forces k = 1, hence f(p) = p.

#### Lemma A. For every odd prime p and every integer k ≥ 2, rad(p^k+1) ≠ rad(p+1).

We split on the parity of k.

**Case k even.** Since p is odd, p ≡ 1 (mod 4) or p ≡ 3 (mod 4); in either case
p² ≡ 1 (mod 8), and as k is even, p^k = (p²)^{k/2} ≡ 1 (mod 8). Hence
p^k + 1 ≡ 2 (mod 8), which gives

  v₂(p^k + 1) = 1.   (†)

(Indeed p^k + 1 is even but not divisible by 4.) Next we compute
gcd(p^k + 1, p + 1). Modulo p + 1 we have p ≡ −1, so p^k ≡ (−1)^k = 1 (k even),
hence p^k + 1 ≡ 2 (mod p + 1). Therefore

  gcd(p^k + 1, p + 1) = gcd(2, p + 1) = 2,

because p + 1 is even (p odd). Consequently **any odd prime dividing p^k + 1 does
not divide p + 1** (a common divisor of p^k+1 and p+1 divides their gcd = 2, so is
a power of 2).

It remains to produce an odd prime dividing p^k + 1. By (†), the integer
(p^k + 1)/2 is odd. Moreover p ≥ 3 and k ≥ 2 give p^k + 1 ≥ 3² + 1 = 10, so
(p^k + 1)/2 ≥ 5 > 1. An odd integer greater than 1 has an odd prime factor q; then
q | p^k + 1 and, by the previous paragraph, q ∤ p + 1. Thus q ∈ rad(p^k + 1) but
q ∉ rad(p + 1), so rad(p^k + 1) ≠ rad(p + 1). This settles the even case.

**Case k odd, k ≥ 3.** Since k is odd, p + 1 divides p^k + 1 with the standard
factorization

  p^k + 1 = (p + 1)·C_k,  where  C_k = Σ_{i=0}^{k−1} (−1)^i p^{k−1−i}
         = p^{k−1} − p^{k−2} + p^{k−3} − ⋯ − p + 1.

(One checks (p+1)·C_k telescopes to p^k + 1; alternatively p^k + 1 = p^k − (−1)^k
and a − b divides aⁿ − bⁿ.)

We first record two facts.

*Fact 1: C_k is odd.* Since p is odd and k is odd, p^k is odd, so p^k + 1 and
p + 1 are both even. Apply **Lifting the Exponent (LTE)** at the prime 2 to
p^k − (−1)^k = p^k + 1 (recall k is odd, so (−1)^k = −1): the odd-exponent form of
LTE states that for an odd exponent n and any prime q dividing a + b,

  v_q(aⁿ + bⁿ) = v_q(a + b) + v_q(n).

With a = p, b = 1, n = k, q = 2 (note 2 | p + 1): v_q(n) = v₂(k) = 0 because k is
odd, so v₂(p^k + 1) = v₂(p + 1). Since p^k + 1 = (p + 1)·C_k, taking v₂ of both
sides gives

  v₂(C_k) = v₂(p^k + 1) − v₂(p + 1) = 0,

so **C_k is odd**.

*Fact 2: C_k > k.* Since k is odd, write k = 2t + 1 with t ≥ 1 (as k ≥ 3). We
regroup C_k exactly. The term of index i = k − 1 in
C_k = Σ_{i=0}^{k−1} (−1)^i p^{k−1−i} is (−1)^{k−1} p^0 = +1 (k − 1 even). Pair the
remaining k − 1 = 2t terms, indices i = 0, …, k − 2, into t consecutive blocks
(i, i+1) with i even; each block contributes

  (−1)^i p^{k−1−i} + (−1)^{i+1} p^{k−2−i} = p^{k−1−i} − p^{k−2−i} = p^{k−2−i}(p − 1),

which is strictly positive. As i runs over 0, 2, …, k − 3, the exponent
k − 2 − i runs over k − 2, k − 4, …, 1, i.e. over the odd numbers 1, 3, …, k − 2.
Hence

  C_k = 1 + Σ_{j=1}^{t} p^{2j−1}(p − 1).

Each summand satisfies p^{2j−1}(p − 1) ≥ p^{1}(p − 1) = p(p − 1) ≥ 3·2 = 6, and
there are t of them, so

  C_k ≥ 1 + 6t.

Since k = 2t + 1, we have 1 + 6t > 1 + 2t = k. Therefore **C_k > k** for all odd
k ≥ 3.

Now the main argument. **Suppose, for contradiction, rad(p^k + 1) = rad(p + 1).**
Let q be any prime dividing C_k. Then q | (p + 1)·C_k = p^k + 1, so by the
assumption rad(p^k + 1) = rad(p + 1) we get q | p + 1. Since C_k is odd, q is odd.
So q is an odd prime with q | p + 1 (equivalently q | p − (−1)). By **Lifting the
Exponent (LTE)** applied to p^k − (−1)^k = p^k + 1 (k odd, q odd, q | p − (−1)):

  v_q(p^k + 1) = v_q(p + 1) + v_q(k).

Since p^k + 1 = (p + 1)·C_k, we have v_q(p^k + 1) = v_q(p + 1) + v_q(C_k), and
subtracting gives

  v_q(C_k) = v_q(k)   for every prime q dividing C_k.

For a prime q **not** dividing C_k we trivially have v_q(C_k) = 0 ≤ v_q(k).
Combining, v_q(C_k) ≤ v_q(k) for **every** prime q. This says exactly that
**C_k divides k**, hence C_k ≤ k. But we proved C_k > k. Contradiction.

Therefore rad(p^k + 1) ≠ rad(p + 1) in the odd case as well, completing Lemma A. ∎

Applying Lemma A to (★): the equality rad(p^k + 1) = rad(p + 1) can only hold with
k = 1, so **f(p) = p for every odd prime p**. ∎ (Step 4)

### Step 5: f(m) = m for all m ∈ ℕ₀

Fix m ∈ ℕ₀. If m = 0, then f(0) = 0 = m by hypothesis, so we are done. Assume
m ≥ 1 and set A = f(m). By Step 1, A ≥ 1 and rad(A) = rad(m).

Apply the FE at (m, p) where p ranges over the odd primes. The two companions are
f(m) + p = A + p and f(p) + m = p + m (using f(p) = p from Step 4). Both are ≥ 1
(indeed ≥ 4). By the Preliminary,

  rad(A + p) = rad(m + p)   for every odd prime p.   (♦)

Note (♦) holds for **all** odd primes p — infinitely many by Euclid — which is
exactly the hypothesis of the following lemma. This is the Step 5 → Lemma B
hand-off: the odd-prime restriction from Step 4 is preserved, and Dirichlet's
choice below will be made among odd primes, where (♦) is available.

#### Lemma B. Let A ≥ 1 and m ≥ 1 be integers with rad(A + p) = rad(m + p) for every odd prime p. Then A = m.

**Proof.** Put D = A − m. Suppose, for contradiction, D ≠ 0.

*Claim: every prime factor of A + p divides D, for every odd prime p.* Let q be a
prime with q | A + p. If q ∤ D, then q ∤ (A + p) − D = m + p (for if q | m + p and
q | A + p then q | (A+p) − (m+p) = D, contradiction). So q | A + p but q ∤ m + p,
i.e. q ∈ rad(A + p) and q ∉ rad(m + p), contradicting rad(A + p) = rad(m + p)
(both A + p, m + p ≥ 1). Hence q | D. This proves the claim.

*Producing a violating prime via Dirichlet.* The integer D·A is nonzero (D ≠ 0 and
A ≥ 1), so it has only finitely many prime divisors; by Euclid's theorem there is a
prime q with q ∤ D·A. In particular q ∤ A, so gcd(−A, q) = gcd(A, q) = 1. By
**Dirichlet's theorem on primes in arithmetic progressions** (knowledge_base.md),
since gcd(−A, q) = 1 the progression

  −A, −A + q, −A + 2q, −A + 3q, …

contains infinitely many primes. Deleting the at most one term equal to the prime
2, we may pick an **odd** prime p with p ≡ −A (mod q). Then

  q | A + p   and   q ∤ D   (since q ∤ D·A).

Thus q is a prime factor of A + p that does **not** divide D, contradicting the
Claim above.

Therefore the assumption D ≠ 0 is untenable, so D = 0, i.e. **A = m**. ∎

Applying Lemma B with A = f(m) (using hypothesis (♦), valid for all odd primes)
yields f(m) = m for every m ≥ 1. Together with f(0) = 0 this gives

  **f(m) = m for all m ∈ ℕ₀.** ∎ (Step 5)

In particular this covers m = 2, which was never resolved via Step 4 (Lemma A only
addresses odd primes and the FE value at p = 2 is never used): Step 5 uses odd
primes p only, and Dirichlet supplies an odd prime p ≡ −A (mod q) as required.

### Step 6: Verification and conclusion

Steps 1–5 show that **any** function f satisfying the hypotheses must be the
identity. It remains to check that f = id actually satisfies the conditions.

- f(0) = 0. ✓
- For all m, n ∈ ℕ₀, f(m) + n = m + n and f(n) + m = n + m = m + n. The two
  numbers are **equal**, hence have the same set of prime divisors, hence are
  companions (equal integers are trivially companions: every prime divides both or
  neither). Equivalently, the companion condition rad(f(m) + n) = rad(f(n) + m)
  becomes rad(m + n) = rad(n + m), which holds trivially. (For the degenerate
  m = n = 0, both sides are 0, and 0 is a companion of 0.) ✓

Therefore f = id is a solution, and by Steps 1–5 it is the only one.

### Answer

The unique function is

  **f(n) = n for all n ∈ ℕ₀** (the identity). ∎

**Theorems invoked:**
- *Lifting the Exponent (LTE)* — for the exact power of a prime dividing p^k + 1
  (Lemma A, odd case, and the v₂ computation).
- *Dirichlet's theorem on primes in arithmetic progressions* — to select an odd
  prime p ≡ −A (mod q) (Lemma B).
- *Euclid's theorem* on the infinitude of primes (Lemma B, to find q ∤ D·A).
