## Full proof

**Answer.** The unique function is **f(n) = n for all n ∈ ℕ₀**.

Throughout, for a positive integer x let rad(x) denote the *set* of prime divisors of x.
We use the convention dictated by the problem's definition of "companion": every prime
divides 0 (vacuously, since a prime p divides 0 iff there is no obstruction — indeed
0 = p·0), so rad(0) is the set of *all* primes. With this convention, two non-negative
integers a, b are companions **iff rad(a) = rad(b)**: "every prime divides both or
neither" says exactly that a prime divides a iff it divides b. In particular a positive
integer is a companion of 0 iff it has every prime as a divisor, which no positive
integer does; so for a, b ∈ ℕ₀ with a, b companions, either both are 0 or both are
positive, and in the positive case rad(a) = rad(b) as sets of primes.

The functional equation hypothesis (FE) is:

> f(0) = 0, and for all m, n ∈ ℕ₀, the numbers f(m)+n and f(n)+m are companions.

---

### Step 1 (Verification: f(n) = n is a solution)

Let f(n) = n. Then f(0) = 0, and for all m, n,
$$ f(m) + n = m + n = n + m = f(n) + m. $$
The two numbers f(m)+n and f(n)+m are *literally equal*, hence have identical sets of
prime divisors, hence are companions. So f(n) = n satisfies the FE. ∎ (existence)

The rest of the proof establishes **uniqueness**: any f satisfying the FE equals the
identity.

Fix, from now on, an arbitrary function f satisfying the FE.

---

### Step 2 (Phase A: substitute m = 0)

Set m = 0 in the FE. Since f(0) = 0, the pair (f(m)+n, f(n)+m) becomes
$$ (f(0)+n,\ f(n)+0) = (n,\ f(n)). $$
Hence n and f(n) are companions, i.e.

> **(A)  rad(f(n)) = rad(n) for all n ∈ ℕ₀.**

Three consequences.

**(A1) f(n) = 0 ⟺ n = 0.** By (A), f(n) and n are companions. As noted above, a
companion of 0 (which has rad = all primes) must itself be 0 among non-negative
integers; and 0 is a companion only of 0. So f(n) = 0 iff n = 0. In particular, **for
every n ≥ 1, f(n) is a positive integer**, so rad(f(n)) is the prime set of a positive
integer (this is what Phases B and D will need).

**(A2) f(1) = 1.** Put n = 1. Then rad(f(1)) = rad(1) = ∅ (the empty set: 1 has no
prime divisors). A non-negative integer x with rad(x) = ∅ must be 1: it cannot be 0
(rad(0) = all primes ≠ ∅), and any integer ≥ 2 has at least one prime divisor, so
rad ≠ ∅. The only remaining possibility is x = 1. Hence **f(1) = 1**.

**(A3) For every prime p, f(p) = pᵃ for some integer a ≥ 1.** Put n = p. Then
rad(f(p)) = rad(p) = {p}. By (A1), f(p) ≥ 1; and rad(f(p)) = {p} means f(p) is a
positive integer whose only prime divisor is p, i.e. **f(p) = pᵃ for some integer
a ≥ 1** (a depends on p).

---

### Step 3 (Phase B: substitute m = 1)

Set m = 1 in the FE. Since f(1) = 1 by (A2), the pair becomes
$$ (f(1)+n,\ f(n)+1) = (n+1,\ f(n)+1). $$
Hence n+1 and f(n)+1 are companions. Both are positive integers (n+1 ≥ 1, and
f(n)+1 ≥ 1), so by the companion characterization,

> **(B)  rad(f(n)+1) = rad(n+1) for all n ∈ ℕ₀.**

---

### Step 4 (Phase C: f(p) = p for every prime p ≥ 3)

Fix a prime p ≥ 3. By (A3), f(p) = pᵃ for some integer a ≥ 1. Applying (B) with n = p:
$$ \operatorname{rad}(p^{a}+1) = \operatorname{rad}(p+1). \tag{$\ast$} $$
We claim a = 1. Suppose, for contradiction, that **a ≥ 2**. We derive a contradiction
to (∗) using Zsygmondy's theorem.

**Key Lemma C.** *For a prime p ≥ 3 and an integer a ≥ 2, one has*
*rad(pᵃ + 1) ≠ rad(p + 1).*

*Proof of Key Lemma C.* Consider the integer
$$ p^{2a} - 1 = (p^{a} - 1)(p^{a} + 1), $$
with exponent n := 2a. Since a ≥ 2 we have **n = 2a ≥ 4**, and p ≥ 3.

We invoke **Zsygmondy's theorem** (knowledge_base.md, "Zsygmondy's theorem (primitive
prime divisors)"), in the form: for an integer p > 1 and exponent n ≥ 1, the number
pⁿ − 1 has a *primitive prime divisor* — a prime r with r ∣ pⁿ − 1 but r ∤ pᵏ − 1 for
all 1 ≤ k < n — **except** in exactly the following cases:

  (i) n = 1 and p − 1 = 1 (i.e. p = 2);
  (ii) n = 2 and p + 1 is a power of 2;
  (iii) (p, n) = (2, 6).

(This is Zsygmondy's theorem for aⁿ − bⁿ specialized to b = 1.) We check that **none**
of the three exceptions applies to our (p, n) = (p, 2a) with p ≥ 3, a ≥ 2:

  • Exception (i) requires n = 1. Here n = 2a ≥ 4 ≠ 1, so (i) does not apply.
  • Exception (ii) requires the exponent n to equal **2**. Here n = 2a ≥ 4 > 2 (because
    a ≥ 2 forces 2a ≥ 4, not merely 2a ≥ 2), so the exponent is not 2, and exception
    (ii) does not apply — regardless of whether p + 1 happens to be a power of 2. (This
    is the boundary the reviewer flagged: even at the smallest case a = 2, n = 4, the
    "base + 1 a power of 2" exception bites only at exponent 2, not at 4, so it is
    excluded.)
  • Exception (iii) requires the base p to equal **2**. Here p ≥ 3, so p ≠ 2 and
    exception (iii) does not apply.

Hence Zsygmondy's theorem guarantees a primitive prime divisor **r** of p^{2a} − 1.
By definition of "primitive,"
$$ r \mid p^{2a} - 1, \qquad r \nmid p^{k} - 1 \ \text{ for all } 1 \le k < 2a. \tag{1} $$
Equivalently (the order characterization in the KB entry), the multiplicative order of
p modulo r is exactly
$$ \operatorname{ord}_r(p) = 2a. \tag{2} $$
Indeed, r ∤ p^{k} − 1 for k < 2a means ord_r(p) does not divide any k < 2a, while
r ∣ p^{2a} − 1 means ord_r(p) ∣ 2a; the smallest positive exponent k with r ∣ p^{k}−1
is therefore 2a, which is ord_r(p). (Note r ∤ p, else r ∣ p^{2a} would contradict
r ∣ p^{2a} − 1, so p is a unit mod r and ord_r(p) is well-defined.)

We now derive two facts about r, *from* (2), without asserting them.

  **(C-i) r ∣ pᵃ + 1.** From (1), r ∣ p^{2a} − 1 = (pᵃ − 1)(pᵃ + 1). Also from (1)
  with k = a (note 1 ≤ a < 2a), r ∤ pᵃ − 1. Since r is prime and divides the product
  (pᵃ − 1)(pᵃ + 1) but not the factor pᵃ − 1, it must divide the other factor:
  r ∣ pᵃ + 1.

  **(C-ii) r ∤ p + 1.** Suppose for contradiction r ∣ p + 1. Then p ≡ −1 (mod r), so
  p² ≡ 1 (mod r), i.e. r ∣ p² − 1. By (2), ord_r(p) = 2a divides every exponent k with
  r ∣ pᵏ − 1; in particular 2a ∣ 2, forcing 2a ≤ 2, i.e. a ≤ 1. This contradicts
  a ≥ 2. Hence r ∤ p + 1.

By (C-i), r ∈ rad(pᵃ + 1); by (C-ii), r ∉ rad(p + 1). Therefore the two prime sets
differ:
$$ \operatorname{rad}(p^{a}+1) \neq \operatorname{rad}(p+1), $$
which proves Key Lemma C. ∎ (Key Lemma C)

Key Lemma C directly contradicts (∗). Hence the assumption a ≥ 2 is impossible, and
a = 1. That is,

> **(C)  f(p) = p for every prime p ≥ 3.**

(The prime p = 2 is *not* treated here and does not need to be — see Step 5.)

---

### Step 5 (Phase D: f(n) = n for all n ∈ ℕ₀)

Suppose, for contradiction, that f(n) ≠ n for some n ∈ ℕ₀. By (A1), n = 0 gives
f(0) = 0 = n, so necessarily **n ≥ 1**. Set
$$ d := f(n) - n. $$
Since f(n) ≠ n, d is a **nonzero integer** (possibly negative); |d| ≥ 1. Note d has only
finitely many prime divisors, and n ≥ 1 has only finitely many prime divisors. Hence we
may choose a prime **q** that divides neither d nor n:
$$ q \nmid d \quad\text{and}\quad q \nmid n, \tag{3} $$
since all but finitely many primes satisfy both conditions. (We may, and do, take q odd,
i.e. q ≥ 3, but this is not essential.)

Because q ∤ n, the residue of −n modulo q is nonzero, so gcd(−n, q) = 1 (q is prime and
does not divide −n). By **Dirichlet's theorem on primes in arithmetic progressions**
(knowledge_base.md, "Dirichlet's theorem on primes in arithmetic progressions"), since
gcd(−n, q) = 1, there are **infinitely many primes p with**
$$ p \equiv -n \pmod{q}. \tag{4} $$
The arithmetic progression {−n + kq : k ≥ 0} contains infinitely many primes; discard
the finitely many that are < 3, and fix one prime p satisfying (4) with **p ≥ 3**. For
this prime, by (C),
$$ f(p) = p. \tag{5} $$

Now apply the FE to the pair (m, n) := (n, p). The two numbers f(n) + p and f(p) + n are
companions. Using (5), f(p) + n = p + n, so:
$$ f(n) + p \ \text{ and }\ p + n \ \text{ are companions.} $$
Both are positive integers (f(n) ≥ 0 and p ≥ 3 give f(n) + p ≥ 3 > 0; and n + p ≥ 4 > 0),
so by the companion characterization,
$$ \operatorname{rad}(f(n) + p) = \operatorname{rad}(p + n). \tag{6} $$

We now exhibit a prime in one radical but not the other, contradicting (6). We claim
**q divides p + n but q does not divide f(n) + p**:

  • **q ∣ (p + n).** By (4), p ≡ −n (mod q), so p + n ≡ 0 (mod q). Hence q ∣ p + n, i.e.
    q ∈ rad(p + n).

  • **q ∤ (f(n) + p).** Write f(n) + p = (n + p) + (f(n) − n) = (n + p) + d. We have
    q ∣ (n + p) from the previous bullet, but q ∤ d by (3). Therefore q ∤ (n + p) + d,
    i.e. q ∤ f(n) + p, i.e. q ∉ rad(f(n) + p). (Concretely: if q divided (n+p)+d, then
    since q ∣ n+p it would divide the difference d, contradicting q ∤ d.)

Thus q ∈ rad(p + n) but q ∉ rad(f(n) + p), so rad(f(n) + p) ≠ rad(p + n), contradicting
(6).

This contradiction shows the assumption "f(n) ≠ n for some n" is untenable. Therefore

> **f(n) = n for all n ∈ ℕ₀.**

---

### Remark on the dependency order (no circularity, no missing base case)

The chain is acyclic: Step 1 (existence) is independent; Steps 2 → 3 → 4 → 5 each use
only earlier steps and the two named theorems. Phase C (Step 4) delivers f(p) = p **only
for primes p ≥ 3**, and Phase D (Step 5) only ever invokes f(p) = p for the *large*
primes produced by Dirichlet (each chosen ≥ 3). So the argument never needs an
independent "f(2) = 2" base case. The values **f(2) = 2 and f(3) = 3 emerge as the n = 2
and n = 3 instances of the conclusion of Step 5**, not as inputs to it; there is no gap
at p = 2. (One could also note f(3) = 3 already from Phase C since 3 ≥ 3, but f(2) = 2
is genuinely first established by Phase D, which is correct.)

---

### Conclusion

The unique function f : ℕ₀ → ℕ₀ with f(0) = 0 such that f(m) + n and f(n) + m are
companions for all m, n is
$$ \boxed{f(n) = n \quad \text{for all } n \in \mathbb{N}_0.} $$
Existence was verified in Step 1 (the two numbers are literally equal); uniqueness was
proved in Steps 2–5. ∎
