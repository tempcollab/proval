## Current best
**Problem reformulation:** The problem is equivalent to constructing a polynomial map F = (P,Q,R): R^3 → R^3 whose image is exactly R^3 \ S.

**Key insight:** The map must satisfy P² + Q² + R² > 0 everywhere BUT with infimum 0 (achieved only "at infinity") to simultaneously avoid origin while covering all nearby points.

**Structural observation:** R^3 \ {point} is NOT homeomorphic to R^3, so any surjective map to it must be non-injective.

The single-point case appears to require semi-algebraic geometry machinery (Fernando-Gamboa-Ruiz type results showing that {f > 0} with f = SOS and finite zeros is a polynomial image). This is research-level and not suitable for an elementary olympiad proof.
