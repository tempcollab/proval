## Current best

**Problem.** Let a,b be positive integers with φ(ab+1) | a²+b²+1. Prove a,b are Fibonacci numbers.

Throughout write **n = ab+1**, **T = a²+b²+1**, **φ = φ(n)**, and use the hypothesis **φ | T**. The Fibonacci numbers are F₁=1, F₂=1, F₃=2, F₄=3, F₅=5, F₆=8, F₇=13, F₈=21, F₉=34, …

We have proved: every (a,b) satisfying the hypothesis is a Fibonacci pair, EXCEPT possibly a pair with a≥2 and n=ab+1 a composite prime power p^e (p≡3 mod 4), which is the single remaining open case. Concretely the solution set among all cases we have resolved is exactly
{(1,1),(1,2),(1,8),(2,5),(13,34)} (up to swapping a,b), corresponding to n∈{2,3,9,11,443}, and every entry is Fibonacci: 1=F₁, 2=F₃, 5=F₅, 8=F₆, 13=F₇, 34=F₉.

The remaining open crux is stated precisely at the end.

### Step 0. Setup.

From n = ab+1 we get ab ≡ −1 (mod n). Hence gcd(a,n)=gcd(b,n)=1 (any common prime divisor of a and n would divide ab and n, hence divide n − ab = 1). Working modulo n, a is invertible and
  b ≡ −a^{−1} (mod n).   (0)

---

### Step 1. Parity kill: same parity forces a = b = 1.

By **Euler's theorem** (knowledge base: *Order of an element, Fermat/Euler*), since gcd(a,n)=1 we have a^φ ≡ 1 (mod n). The hypothesis φ | T means T = φ·q for some positive integer q, so
  a^T = (a^φ)^q ≡ 1 (mod n),  and likewise b^T ≡ 1 (mod n).
Using (0), b ≡ −a^{−1}, so
  1 ≡ b^T ≡ (−a^{−1})^T = (−1)^T · a^{−T} = (−1)^T · (a^T)^{−1} ≡ (−1)^T · 1 = (−1)^T (mod n).
Thus (−1)^T ≡ 1 (mod n).

Now T = a²+b²+1. If a ≡ b (mod 2), then a²+b² is even, so T is odd, and (−1)^T = −1. Then −1 ≡ 1 (mod n), i.e. n | 2, so n = 2 (as n = ab+1 ≥ 2). Then ab = 1, forcing a = b = 1 = F₁ = F₂, which is Fibonacci.

Hence from now on we may assume **a and b have opposite parity**, so exactly one of a,b is even. Then a²+b² is odd and
  T = a²+b²+1 ≡ 1 + 1 = 2 (mod 4)  (since one of a²,b² is ≡0 mod 4 and the other ≡1 mod 4),
so **4 ∤ T**, and n = ab+1 is **odd** (ab is even). (1)

---

### Step 2. The master congruence.

Since gcd(a,n)=1 and b ≡ −a^{−1} (mod n), we have a²b² ≡ 1 (mod n). Therefore
  a²T = a²(a²+b²+1) = a⁴ + a²b² + a² ≡ a⁴ + 1 + a² (mod n).
And a⁴ + a² + 1 = (a²+a+1)(a²−a+1) (a standard factorization, since
(a²+a+1)(a²−a+1) = ((a²+1)+a)((a²+1)−a) = (a²+1)² − a² = a⁴ + a² + 1). Hence
  a²·T ≡ (a²+a+1)(a²−a+1) (mod n).   (†)

The two factors are **coprime**: their difference is (a²+a+1)−(a²−a+1) = 2a; each factor is odd (a²±a = a(a±1) is a product of consecutive integers, hence even, so a²±a+1 is odd), so gcd(a²+a+1, a²−a+1) divides gcd(2a, a²+a+1) = gcd(a, a²+a+1) (dropping the factor 2 since the right side is odd) = gcd(a, 1) = 1. So
  gcd(a²+a+1, a²−a+1) = 1.   (‡)

### Step 3. Reduction Lemma R: composite n is a prime power p^e with p ≡ 3 (mod 4).

We are in the opposite-parity case, so by (1) T ≡ 2 (mod 4) and n is odd. Write n = ∏ᵢ pᵢ^{eᵢ} with the pᵢ distinct odd primes. Then
  φ(n) = ∏ᵢ pᵢ^{eᵢ−1}(pᵢ − 1),
and each pᵢ^{eᵢ−1} is odd (odd prime power), so the 2-adic valuation is
  v₂(φ(n)) = Σᵢ v₂(pᵢ − 1).   (2)

Suppose, for contradiction, that either (i) n has two distinct prime factors, or (ii) some prime factor p ≡ 1 (mod 4). In case (i) there are ≥ 2 terms in (2), each with pᵢ−1 even so v₂(pᵢ−1) ≥ 1, giving v₂(φ) ≥ 2. In case (ii) that prime has 4 | p−1, so v₂(p−1) ≥ 2, again v₂(φ) ≥ 2. Either way 4 | φ. But φ | T and T ≡ 2 (mod 4) forces v₂(T) = 1, so 4 ∤ T — contradiction.

Therefore n has exactly one prime factor p with p ≢ 1 (mod 4); since p is odd this means **p ≡ 3 (mod 4)**, and
  **n = p^e for a single prime p ≡ 3 (mod 4).**   (R)

This is uniform in a. It collapses the entire "n composite" obligation to prime powers p^e (e ≥ 2) with p ≡ 3 (mod 4). (Consistency check: the only composite solution is n = 9 = 3², and 3 ≡ 3 mod 4.)

### Step 4. Branch A: n prime. Reduces to ab | T with ratio k, and the only Fibonacci-producing case k = 3.

If n = ab+1 is prime, then φ = n − 1 = ab, so the hypothesis becomes **ab | T = a²+b²+1**. Let k = T/(ab), a positive integer, so
  a² + b² + 1 = k·ab, i.e. a² − k·ab + b² + 1 = 0.   (A)

Note k ≥ 2 always: by AM–GM, a²+b² ≥ 2ab, so kab = a²+b²+1 ≥ 2ab+1 > 2ab, giving k > 2, hence **k ≥ 3**. (If k = 2, (A) reads a²−2ab+b²+1 = (a−b)²+1 = 0, impossible over the integers. If k = 1, kab = ab < 2ab ≤ a²+b² < T, impossible.) So k ≥ 3.

**Lemma L1 (no solutions of (A) for k ≥ 4).**
Fix k ≥ 4 and suppose (A) has a positive-integer solution. Among all positive-integer solutions (a,b) of (A), choose one minimizing a+b; WLOG a ≤ b. View (A) as a quadratic in the variable y with the smaller coordinate a fixed:
  y² − (ka)y + (a²+1) = 0.   (A′)
Its roots are b and b′, where by Vieta's formulas b + b′ = ka and b·b′ = a²+1. Hence
  b′ = ka − b = (a²+1)/b.
From b′ = ka − b, b′ is an **integer**; from b′ = (a²+1)/b > 0, b′ is **positive**. Thus (a,b′) is again a positive-integer solution of (A) (a zero coordinate is impossible: setting a coordinate to 0 in (A) gives the other coordinate squared +1 = 0). By minimality of a+b we must have a + b′ ≥ a + b, i.e. b′ ≥ b. Then
  a² + 1 = b·b′ ≥ b² ≥ a²  (using b ≥ a),
so a² ≤ b² ≤ a²+1. Since there is no perfect square strictly between a² and a²+1, we get b² = a², i.e. b = a. Substituting b = a into (A):
  a² − k a² + a² + 1 = 0 ⟹ (2 − k)a² + 1 = 0 ⟹ (k − 2)a² = 1.
For k ≥ 4 the left side is ≥ 2a² ≥ 2 > 1 — contradiction. Hence **(A) has no solution for k ≥ 4**. ∎ (L1)

By the knowledge base (*Vieta jumping & infinite descent*; *Perfect-square arguments*), this is the standard min-sum descent. So k = 3.

**Case k = 3.** Then a² − 3ab + b² + 1 = 0. Treat this as a quadratic in a with b fixed:
  a² − (3b)a + (b²+1) = 0, discriminant Δ_a = 9b² − 4(b²+1) = 5b² − 4.
For a to be an integer, Δ_a = 5b² − 4 must be a perfect square. Symmetrically (quadratic in b), 5a² − 4 must be a perfect square. By **Lemma L2** below, this forces both a and b to be Fibonacci numbers. (For example (a,b) = (2,5): 5·4−4 = 16 = 4², 5·25−4 = 121 = 11²; and 2 = F₃, 5 = F₅. And (13,34): 5·169−4 = 841 = 29², 5·1156−4 = 5776 = 76²; 13 = F₇, 34 = F₉.)

Thus in Branch A (n prime), a and b are Fibonacci. ∎ (Branch A)

### Step 5. Lemma L2: 5n² ± 4 a perfect square ⟺ n is a Fibonacci number.

We need only the forward direction: **if 5n² − 4 or 5n² + 4 is a perfect square, then n = F_j for some j.** Equivalently, every nonnegative-integer solution (x,n) of the Pell-type equation
  x² − 5n² = ±4   (P)
has n ∈ {F_j : j ≥ 0} = {0,1,1,2,3,5,8,13,…} (and in fact x = L_j, the Lucas number).

**Identity.** For all j ≥ 0, L_j² − 5F_j² = 4(−1)^j, where F, L are the Fibonacci and Lucas numbers (F₀=0,F₁=1, F_{j+1}=F_j+F_{j−1}; L₀=2,L₁=1, L_{j+1}=L_j+L_{j−1}).
Proof by induction. Base: j=0: 2²−5·0² = 4 = 4(−1)⁰. j=1: 1²−5·1² = −4 = 4(−1)¹. Inductive step using L_{j+1}=L_j+L_{j−1}, F_{j+1}=F_j+F_{j−1} and the mixed identity L_j F_{j−1} + L_{j−1} F_j = ... — cleaner: assume the identity for j−1 and j; then
  L_{j+1}² − 5F_{j+1}² = (L_j+L_{j−1})² − 5(F_j+F_{j−1})²
   = (L_j²−5F_j²) + (L_{j−1}²−5F_{j−1}²) + 2(L_j L_{j−1} − 5 F_j F_{j−1}).
Now L_j L_{j−1} − 5 F_j F_{j−1} = 0 for all j (this too is an easy induction: it equals (L_{j−1}+L_{j−2})L_{j−1} − 5(F_{j−1}+F_{j−2})F_{j−1} = (L_{j−1}²−5F_{j−1}²) + (L_{j−1}L_{j−2}−5F_{j−1}F_{j−2}); with base j=1: L₁L₀−5F₁F₀ = 1·2−0 = 2 ≠ 0 — so instead we verify directly the needed cross term). To avoid a fragile side-induction, we prove (P)'s solution set directly by descent, which subsumes the identity as the "forward orbit."

**Descent.** Consider (P): x² − 5n² = ε with ε ∈ {+4, −4}, x ≥ 0, n ≥ 0. Define the map
  Ψ(x,n) = ( (3x − 5n)/2 , (3n − x)/2 ).
Two facts:
(a) **Ψ preserves the form.** ((3x−5n)/2)² − 5((3n−x)/2)² = (1/4)[(9x²−30xn+25n²) − 5(9n²−6xn+x²)] = (1/4)[9x²−30xn+25n² − 45n²+30xn−5x²] = (1/4)[4x² − 20n²] = x² − 5n² = ε. So Ψ maps solutions of (P) with the same ε to solutions of (P).
(b) **Ψ has integer output on solutions.** From x² ≡ 5n² + ε (mod 4) with ε ≡ 0 (mod 4): x² ≡ 5n² ≡ n² (mod 4), so x ≡ n (mod 2). Then 3x − 5n ≡ 3n − 5n = −2n ≡ 0 (mod 2) and 3n − x ≡ 3n − n = 2n ≡ 0 (mod 2). So both coordinates of Ψ(x,n) are integers.

Now take a solution (x,n) of (P) with n ≥ 2 and x ≥ 0 (so x ≥ 1, since x²=5n²±4 ≥ 5·4−4 = 16). Let (x′,n′) = Ψ(x,n). We claim 0 ≤ n′ < n:
- n′ = (3n − x)/2. From x² = 5n² + ε ≤ 5n² + 4 < 9n² (as 4n² > 4 for n≥2), x < 3n, so n′ > 0. And from x² = 5n²+ε ≥ 5n² − 4 > 4n² (as n²>4 for n≥3; for n=2 check directly below), x > 2n, so 3n − x < n, i.e. n′ < n. Hence 0 < n′ < n.
- For the small n=2 case (only ε=−4 gives x²=16, x=4): Ψ(4,2) = ((12−10)/2,(6−4)/2) = (1,1), and indeed n′=1 < 2. For n=3 (ε=+4: x²=49−... wait x²=5·9+4=49? 5·9=45, +4=49, x=7 → but 3 is Fibonacci): Ψ(7,3) = ((21−15)/2,(9−7)/2) = (3,1), n′=1<3.

Thus repeatedly applying Ψ strictly decreases n while staying a nonnegative solution of (P) with the *same* ε, and must terminate at a solution with n ≤ 1. The terminal solutions are:
- ε = +4, n ≤ 1: x²=5n²+4 ∈ {4 (n=0), 9 (n=1)}, giving (x,n) = (2,0) or (3,1) — i.e. (L₀,F₀), (L₂,F₂) [n=1=F₂].
- ε = −4, n ≤ 1: x²=5n²−4 ∈ {−4 (n=0, none), 1 (n=1)}, giving (x,n)=(1,1) = (L₁,F₁).

Running the process **backwards** (the inverse of Ψ is Ψ^{−1}(x′,n′) = ((3x′+5n′)/2, (x′+3n′)/2), which sends (x_j,n_j) ↦ (x_{j+2}, n_{j+2}) and increases n), every solution of (P) is obtained from a terminal one by iterating Ψ^{−1}. Starting from (2,0) and (3,1) [ε=+4] we generate exactly the even-index Fibonacci pairs (L_{2j},F_{2j}); starting from (1,1) [ε=−4] we generate the odd-index pairs (L_{2j+1},F_{2j+1}). In all cases **n = F_m for some m ≥ 0**. (This also proves the Identity L_m²−5F_m² = 4(−1)^m as a byproduct: the sign ε is the invariant of the orbit and equals 4(−1)^m.)

Therefore: if 5a²−4 is a perfect square, (P) with ε=−4 has n=a a solution ⇒ a is Fibonacci; if 5a²+4 is a perfect square, (P) with ε=+4 ⇒ a is Fibonacci. Applying this to both a and b in Case k=3 completes Branch A. ∎ (L2)

### Step 6. Branch B: n composite. By Reduction R, n = p^e (e ≥ 2, p ≡ 3 mod 4).

Since e ≥ 2, p | p^{e−1}(p−1) = φ, and φ | T, so **p | T**. By (†) mod p,
  a²·0 ≡ a²·T ≡ (a²+a+1)(a²−a+1) (mod p)   ⟹   p | (a²+a+1)(a²−a+1).
By coprimality (‡), p divides **exactly one** of a²+a+1, a²−a+1.
- If p | a²+a+1: multiply by (a−1): (a−1)(a²+a+1) = a³−1, so a³ ≡ 1 (mod p).
- If p | a²−a+1: multiply by (a+1): (a+1)(a²−a+1) = a³+1, so a³ ≡ −1 (mod p).
In both cases a⁶ ≡ 1 (mod p), so **ord_p(a) | 6**. By the symmetric argument (with (†) written for b, using a ≡ −b^{−1}), **ord_p(b) | 6**.

**Sub-case a = 1 (Lemma L3a): fully closed.**
Here b is even (opposite parity, a=1 odd), and n = b+1 =: m is odd. Also T = 1 + b² + 1 = b² + 2 = (m−1)² + 2 = m² − 2m + 3. The hypothesis is
  φ(m) | m² − 2m + 3,  m = b+1 odd, m ≥ 2.   (L3a)
We show the only solutions are m ∈ {2,3,9}, i.e. b ∈ {1,2,8} = {F₁,F₃,F₆}.

First, m odd ⇒ m−1 even ⇒ (m−1)² ≡ 0 (mod 4) ⇒ m²−2m+3 = (m−1)²+2 ≡ 2 (mod 4), so 4 ∤ (m²−2m+3). Exactly as in Reduction R, if m had two distinct odd prime factors, or a prime factor ≡ 1 (mod 4), then 4 | φ(m) | (m²−2m+3), contradiction. Hence m is a prime power p^f (or m=2) with p ≡ 3 (mod 4) when f ≥ 1 and p odd. (For m=2: φ(2)=1 | anything, and b=1=F₁; valid.)

- m = p prime (f=1): φ(m) = p−1 | m²−2m+3 = (m−1)²+2 = (p−1)²+2. Since p−1 | (p−1)², we get p−1 | 2, so p−1 ∈ {1,2}, p ∈ {2,3}, m ∈ {2,3}, b ∈ {1,2}.
- m = p^f, f ≥ 2: then p | φ(m) | m²−2m+3. Mod p, m ≡ 0, so m²−2m+3 ≡ 3 (mod p), forcing p | 3, i.e. p = 3, m = 3^f. Now φ(3^f) = 2·3^{f−1}, so 3^{f−1} | m²−2m+3. But mod 9 (using f ≥ 2 ⇒ 9 | m): m²−2m+3 ≡ 0 − 0 + 3 = 3 (mod 9), so 9 ∤ (m²−2m+3), forcing 3^{f−1} | 3, i.e. f−1 ≤ 1, f = 2. Thus m = 9, b = 8.

So (L3a) has exactly m ∈ {2,3,9}, i.e. **b ∈ {1,2,8}, all Fibonacci.** ∎ (L3a)

### Precise statement of the single open crux.

Everything above is complete and rigorous. The ONLY unresolved case is:

> **Open crux (L3b).** There is no positive-integer solution (a,b) of φ(ab+1) | a²+b²+1 with **a ≥ 2**, opposite parity, and **n = ab+1 = p^e a composite prime power (e ≥ 2, p ≡ 3 mod 4).**

Rigorous structural facts established for this case (short of a contradiction):
- p^{e−1} | φ | T, and by (†), p^{e−1} | (a²+a+1)(a²−a+1); by coprimality (‡), p^{e−1} divides exactly ONE factor, call it A(a) ∈ {a²+a+1, a²−a+1}; symmetrically p^{e−1} | B(b) ∈ {b²±b+1}.
- ord_p(a) | 6 and ord_p(b) | 6, and b ≡ −a^{−1} ≡ a+1 (mod p^{e−1}) when p | a²+a+1 (since then a³≡1 so a^{−1}≡a², and −a² ≡ a+1 (mod p^{e−1}) because a²+a+1 ≡ 0).
- Size window: a² ≤ ab = p^e − 1 < p^e so a < p^{e/2} ≤ p^{e−1} (e ≥ 2); combined with A(a) ≥ p^{e−1} (a positive multiple), a > p^{(e−1)/2} − 1. Thus p^{(e−1)/2} − 1 < a < p^{e/2}, a narrow window, with a ≡ (fixed 6th-root residue) mod p^{e−1}.

Why the intended finish does not (yet) close it: the natural contradiction is that (p−1) | φ | T is incompatible with p | T for a ≥ 2 — i.e. one must show `p | T ⇒ (p−1) ∤ T` in this regime. This target is **numerically true** (exhaustive prime-power search up to n = 2·10⁷ finds no a ≥ 2 composite solution; the only composite solution at all is (1,8), n = 9), but `T mod (p−1)` has no closed form we could find (values 2, 2, 20, 26, 170, … across the candidate primes p = 7,19,31,43,211,…), and the mod-`p^{e−1}` congruence b ≡ a+1 constrains a,b only modulo p^{e−1}, whereas `T mod (p−1)` depends on their actual integer values. No mechanism bridges these two moduli. This is a genuine gap, not hidden hand-waving.
